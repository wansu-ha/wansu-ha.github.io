노노그램을 풀어 영토를 큐레이션하는 협력 모자이크 웹게임. 노노그램 풀이 엔진, 토러스 무한 월드, 이중 통화 경제 등 알고리즘·게임 수학·동시성까지 직접 다뤘다. 기획·아키텍처·기술 선택은 직접 결정하고, 세부 구현은 AI 페어로 가속했다.

핵심 도전

노노그램 풀이·출제 검증

5~10 가변 화판의 풀이 수를 판정(2~5개는 연작 흡수). 라인 시뮬레이션으로 난이도 별점(1~5)·라벨(universal·maniac·reject)을 매기고, 거부 시 1셀 자동수정을 제안. 전역 중복은 canonical SHA-256으로 차단.

solveAll(hints, size, 6) // 1=단일·2~5=연작·6+=거부 (size 5~10) // 출제검증(PVS) p99: 5×5<5ms·10×10<100ms · CI 회귀 게이트

UPDATE ... SET bal = bal - :n WHERE bal >= :n -- 0행이면 롤백
